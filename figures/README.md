# Figures Directory

Place your model architecture figures here.

## Expected Files

- `mehe_architecture.png` - Main MEHE architecture diagram

## Alternative Paths

If you prefer to place figures elsewhere, you can update the README.md to reference:
- `./mehe_architecture.png` (in root directory)
- `./images/mehe_architecture.png` (in images directory)
- Or any other path relative to the README.md file

## Supported Formats

GitHub supports the following image formats in README files:
- PNG (recommended for diagrams)
- JPG/JPEG
- GIF
- SVG (for scalable vector graphics)
